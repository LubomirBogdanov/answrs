void uart_init ( uint32_t  baud_rate)

This function initializes the UART module. It can be used for redirecting the printf function.

To initialize the UART, first the clock of the module has to be enabled. Second, the GPIO pins have to be multiplexed to the UART module. Third, the module has to be enabled and last - the UART baudrate has to be set according to the user settings.
