void led_init ( void  )

This function initializes a single pin from a GPIO port where an LED is connected.

To do an init, first the clock to the GPIO module has to be enabled. Second, the respective GPIO has to be configured as an OUTPUT. Third, the output must be configured to be a push-pull. Finally, the GPIO is turn ON upon initialization.

Returns
None.
