This function initializes a single pin from a G\-P\-I\-O port where an L\-E\-D is connected. 

To do an init, first the clock to the G\-P\-I\-O module has to be enabled. Second, the respective G\-P\-I\-O has to be configured as an O\-U\-T\-P\-U\-T. Third, the output must be configured to be a push-\/pull. Finally, the G\-P\-I\-O is turn O\-N upon initialization.

\begin{DoxyReturn}{Returns}
None.
